Section 6.1
What is a Neural Network?
A network of neurons is called a Neural Network. "Neural" comes from the word neuron of the human nervous system — the basic unit of the brain used to process and transmit information to all other nerve cells and muscles. Artificial Intelligence embeds a similar behaviour of neurons into a network called an Artificial Neural Network (ANN).
🧬

Inspired by Biology

ANNs mimic the way biological neurons work together to identify phenomena, weigh options, and arrive at conclusions — just like the human brain.

🔄

Adaptive

A neural network can adapt to changing inputs, generating the best possible outcome without any changes to the program — exactly as the human brain does.

🤖

Automatic Feature Learning

The main advantage of ANNs is that they can extract data features automatically without needing any manual input from the programmer.

🌐

Real-World Applications

Auto-reply emails, spam filtering, Facebook image tagging, e-shopping recommendations, chatbots, and Google's search algorithm all use neural networks.

6.1.1 Parts of a Neural Network

Every neural network comprises layers of interconnected nodes — an input layer, hidden layer(s), and an output layer.

Neural Network Architecture — Layer Structure

1
Input Layer

Consists of units representing the input fields. Each unit corresponds to a specific feature or attribute of the problem being solved (e.g., X₁, X₂, X₃).

2
Hidden Layer(s)

Located between the input and output layers. Each hidden layer contains nodes/artificial neurons which process the input data. Nodes are interconnected with associated weights. A network with 2+ hidden layers is called a Deep Neural Network.

3
Output Layer

Consists of one or more units representing the target field(s). The output units generate the final predictions or outputs of the neural network (e.g., ŷ = predicted value).

📘 Deep Learning: An ANN with two or more hidden layers is known as a Deep Neural Network. Training such networks is called Deep Learning. The term "deep" refers to the number of hidden layers. A network with more than 3 layers total (including input and output) is considered a deep learning algorithm.
Section 6.2
Components of a Neural Network
The key components of a neural network work together to process information, learn patterns, and generate predictions. Understanding each component is essential to understanding how neural networks are trained and optimized.

Neural Network Components — Overview Diagram

1. Neurons (Nodes)

  • Fundamental building blocks of a neural network
  • Receive inputs from other neurons or external sources
  • Compute weighted sum of inputs, apply activation function
  • Produce an output passed to the next layer

2. Weights

  • Represent the strength of connections between neurons
  • Each connection (synapse) has an associated weight
  • Convey the importance of that feature in predicting the output
  • Network learns optimal weights during training to minimize error

3. Activation Functions

  • Act as decision-makers for each neuron
  • Decide whether a neuron should activate (send signal)
  • Types: Sigmoid, Tanh, ReLU (Rectified Linear Unit)
  • Add non-linearities, allowing networks to learn complex patterns

4. Bias

  • Constants added to the weighted sum before activation
  • Allow the network to shift the activation function horizontally
  • Help account for any inherent bias in the data
  • Every neuron has an associated bias term

5. Connections

  • Represent the synapses between neurons
  • Each connection has an associated weight determining its influence
  • Biases (constants) are also associated with each neuron
  • Together they define the network's decision boundary

6. Learning Rule

  • Specifies how weights and biases are adjusted during training
  • Backpropagation computes gradients and updates weights
  • Goal: minimize the network's prediction error
  • Common optimizer: Gradient Descent

7. Propagation Functions

  • Forward Propagation: Input data flows through layers, activations computed, output compared to target (loss)
  • Back Propagation: Gradients propagated backward to update weights using optimization algorithms (e.g., gradient descent)

Forward Propagation vs. Back Propagation

⚡ Backpropagation: Backpropagation (short for "backward propagation of errors") is an optimization algorithm used during neural network training. It adjusts the weights of the network based on the error (loss) obtained in the previous iteration (epoch). Proper tuning ensures lower error rates, making the model more reliable.
Section 6.3
Working of a Neural Network
Imagine each node as a simple calculator. It takes input numbers, multiplies them by certain values (weights), adds them together, adds an extra number (bias), and then gives an output. This output is used as input for the next node in the network.
Neuron Output Formula
∑wᵢxᵢ + bias = w₁x₁ + w₂x₂ + w₃x₃ + bias
f(x) = 1, if ∑wᵢxᵢ + b ≥ 0 (Activated)
f(x) = 0, if ∑wᵢxᵢ + b < 0 (Inactive)
Each input is assigned a weight to show its importance. The total is passed through an activation function to decide if the node should "fire" and pass data to the next layer.

x1, x2, x3 = 2, 3, 1
w1, w2, w3 = 0.4, 0.2, 0.6
bias = 0.1
threshold = 3.0

output = w1*x1 + w2*x2 + w3*x3 + bias
# = (0.4*2) + (0.2*3) + (0.6*1) + 0.1

# 2.1 < 3.0 → Output = 0 (INACTIVE)

Section 6.4
Types of Neural Networks
Neural networks can be classified into different types, used for different purposes. Here are the most common types you will encounter and their key use cases.

Neural Network Types — Classification Tree

⚡
6.4.1 · Standard Neural Network

Perceptron

The perceptron, created by Frank Rosenblatt in 1958, is a simple neural network with a single layer of input nodes fully connected to a layer of output nodes. It uses Threshold Logic Units (TLUs) as artificial neurons.

🎯 Application: Binary classification, spam detection, basic decision-making
→
6.4.2 · Feed Forward Neural Network

FFNN / Multi-Layer Perceptron (MLP)

FFNNs have an input layer, one or more hidden layers, and an output layer. Data flows only in one direction — from input to output. They use activation functions and weights to process information forward. Efficient for handling noisy data and relatively straightforward to implement.

🎯 Application: Image recognition, NLP, regression tasks
🔬
6.4.3 · Convolutional Neural Network

CNN

CNNs utilize filters to extract features from images, enabling robust recognition of patterns and objects. They incorporate a three-dimensional arrangement of neurons particularly effective for processing visual data.

🎯 Application: Object detection, image recognition, style transfer, medical imaging
🔁
6.4.4 · Recurrent Neural Network

RNN

RNNs are designed for sequential data and feature feedback loops to allow information persistence across time steps. If the prediction is wrong, the learning rate makes small adjustments, gradually improving prediction during backpropagation.

🎯 Application: Language modeling, machine translation, chatbots, speech recognition, time series
🎨
6.4.5 · Generative Adversarial Network

GAN

GANs consist of two neural networks — a generator (creates new data instances) and a discriminator (evaluates authenticity) — trained simultaneously. Used for unsupervised learning and generating realistic synthetic data.

🎯 Application: Image generation, style transfer, data augmentation

Problem: Train a neural network to learn the formula f = c × 1.8 + 32 without explicitly programming it — only from example data.

celsius_to_fahrenheit.py
# Activity 2: Celsius → Fahrenheit with TensorFlow
import tensorflow as tf
import numpy as np
import matplotlib.pyplot as plt

# ── Training Data ──────────────────────────────────
c = np.array([-40, -10, 0, 8, 15, 22, 38], dtype=float)
f = np.array([-40, 14, 32, 46, 59, 72, 100], dtype=float)

# ── Create Model ───────────────────────────────────
# Single layer, single neuron — problem is simple
model = tf.keras.Sequential([
    tf.keras.layers.Dense(
        units=1,
        input_shape=[1]   # Single input value (Celsius)
    )
])

# ── Compile Model ──────────────────────────────────
# loss: measures how far predictions are from actual values
# optimizer: adjusts weights to reduce the loss
model.compile(
    loss='mean_squared_error',
    optimizer=tf.keras.optimizers.Adam(0.1),
    metrics=['mean_squared_error']
)

# ── Train Model ────────────────────────────────────
history = model.fit(c, f, epochs=500, verbose=False)
print("✅ Finished training the model")

# ── Training Loss Graph ─────────────────────────── 
plt.xlabel('Epoch Number')
plt.ylabel('Loss Magnitude')
plt.plot(history.history['loss'])
plt.title('Training Loss over Epochs')
plt.show()

# ── Predict Values ──────────────────────────────── 
predicted = model.predict(np.array([100.0]))
print(f"NN prediction for 100°C: {predicted[0][0]:.2f}°F")

# ── Compare with Formula ────────────────────────── 
f_formula = (100 * 1.8) + 32
print(f"Formula result for 100°C: {f_formula}°F")  # 212.0

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import tensorflow as tf

# ── Step 1: Load Dataset ────────────────────────── 
temp_df = pd.read_csv('cel_fah.csv')
print(temp_df.head())

# ── Step 2: Visualise Data ──────────────────────── 
plt.scatter(temp_df['Celsius'], temp_df['Fahrenheit'])
plt.xlabel('Celsius'); plt.ylabel('Fahrenheit')
plt.title('Celsius vs Fahrenheit'); plt.show()

# ── Step 3: Assign Variables ────────────────────── 
X_train = temp_df['Celsius']
y_train = temp_df['Fahrenheit']

# ── Step 4: Define Sequential ANN ──────────────── 
model = tf.keras.Sequential()

model.add(tf.keras.layers.Dense(units=32, input_shape=(1,)))
# Second hidden layer (32 nodes)
model.add(tf.keras.layers.Dense(units=32))

# ── Step 5: Model Summary ───────────────────────── 
# Shows: layer order, output shapes, number of parameters
model.summary()

# ── Step 6: Compile ─────────────────────────────── 
model.compile(
    optimizer=tf.keras.optimizers.Adam(.1),
    loss='mean_squared_error'
)

# ── Step 7: Train (30 epochs, 20% validation) ───── 
epochs_hist = model.fit(X_train, y_train,
                        epochs=30,
                        validation_split=0.2)

# ── Step 8: Predict (Deployment) ────────────────── 
Temp_C = float(input("Enter temperature in Celsius: "))
Temp_F = model.predict(np.array([Temp_C]))
print(f"Fahrenheit (ANN)  = {Temp_F[0][0]:.2f}")

B · Short Answer Questions
Q1. What is the purpose of an activation function in a neural network?
An activation function decides whether a neuron should be "activated" (send a signal) or not, based on the weighted sum of its inputs. It introduces non-linearity into the network, which allows the neural network to learn and model complex, non-linear relationships in the data. Common types include Sigmoid, Tanh, and ReLU.
Q2. How does backpropagation work in a neural network?
Backpropagation works in two phases: (1) Forward pass — input data flows through the network, activations are computed, and the output is compared to the true value to calculate loss/error. (2) Backward pass — gradients of the loss are computed with respect to each weight, and the weights are updated using an optimization algorithm (e.g., gradient descent) to minimize the error. This process repeats over many epochs until the model converges.
Q3. What are the key components of a neural network?
The key components are: (1) Neurons — basic computing units, (2) Weights — representing connection strengths, (3) Biases — constants that shift activation, (4) Activation Functions — introduce non-linearity, (5) Connections — synapses linking neurons, (6) Learning Rule — specifies how weights are updated, and (7) Propagation Functions — forward and backward propagation to train the network.
Q4. Describe the structure of a feedforward neural network.
A feedforward neural network (FFNN) consists of: (1) Input Layer — receives raw features, (2) one or more Hidden Layers — process data using weights, biases, and activation functions, and (3) an Output Layer — produces the final prediction. Data flows in one direction only — from input to output — without any feedback loops. FFNNs are also known as Multi-Layer Perceptrons (MLPs).
Q5. What are the differences between FFNN and RNN?
FFNN: Data flows in one direction (input → output), no memory between inputs, suitable for static data like images. RNN: Has feedback loops allowing information to persist across time steps, maintains hidden state (memory), suitable for sequential data like text, speech, and time series. RNNs can relate current inputs to previous context, making them powerful for language modeling and machine translation.
D · Competency Based Questions
CBQ 1. You are a data scientist developing a neural network to predict heart attack risk. Which type of neural network model would you build and why?
Model Choice: Feedforward Neural Network (MLP) — Because the dataset contains structured tabular features (age, gender, blood pressure, cholesterol, etc.) with no sequential or image data. Input layer: features like age, BP, cholesterol. Hidden layers: 2-3 layers to learn non-linear relationships between risk factors. Output layer: single neuron with Sigmoid activation → probability of heart attack (0 to 1). Activation: ReLU for hidden layers, Sigmoid for output. Loss: Binary Cross-Entropy. Optimizer: Adam.
CBQ 2. You are deciding between two restaurants considering: food quality, ambience, and distance. Using a neural network concept, determine the inputs and weights.
Inputs: X₁ = Food Quality (rate 0-1), X₂ = Ambience (rate 0-1), X₃ = Distance (1 = close, 0 = far). Weights: W₁ = 5 (food quality is most important), W₂ = 3 (ambience is moderately important), W₃ = 2 (distance matters but less). Bias = −3 (threshold). Apply formula: ŷ = (X₁×5) + (X₂×3) + (X₃×2) − 3. If ŷ ≥ threshold → choose that restaurant (output = 1).
